One Sentence Description *
Verbinden Sie Ihre NodeMCU-, WeMos mini D1-, ESP8266-, ESP32-, Arduino- und Raspberry Pi-Entwicklungsboards kostenlos mit Amazon Alexa!

Detailed Description *
Mit Sinric Pro können Sie Ihre NodeMCU, WeMos mini D1, ESP8266, ESP32, Arduino, Raspberry Pi oder beliebige IOT-Entwicklungsboards KOSTENLOS mit Alexa Control verbinden!

Wie benutzt man?

1. Gehen Sie zu https://sinric.pro.

2. Erstellen Sie ein Konto und melden Sie sich an

3. Erstellen Sie ein neues Gerät. zB: tv

Verwenden Sie die auf dem Startbildschirm angezeigten Anmeldeinformationen, um eine Verbindung zu Ihrer Entwicklungsplatine herzustellen.

Hier ist ein Beispiel, wie man eine Verbindung herstellt:
https://github.com/kakopappa/sinric/blob/master/arduino_examples/switch_example.ino

Ersetzen Sie Ihre Anmeldeinformationen, "Ihre WLAN-SSID", "Ihr WLAN-Passwort" im Code.

4. Installieren / Aktivieren Sie die Fertigkeit Sinric Smart Home

5. Bitten Sie Alexa, Geräte zu erkennen.

6. Sag Alexa, schalte "tv" ein


Example Phrases*
Alexa, schalte den Fernseher ein
